# macOS KeyBindings (按键绑定)

## 数字键 (Numerical Keypad Keys)

|前缀|解释|
|:---|:---|
|~|⌥ Option |
|$|⇧ Shift |
|^|^ Contro|
|@|⌘ Command |
|#|数字键|
