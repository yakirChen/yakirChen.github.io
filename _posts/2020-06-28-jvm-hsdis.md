jdk1.8
```bash
cd jdk8u/hotspot/src/share/tools/hsdis/
make BINUTILS=binutils-2.31 ARCH=amd64
```


jdk11:
```bash
cd jdk11ubuild/src/utils/hsdis
make BINUTILS=binutils-2.31 ARCH=amd64
```
